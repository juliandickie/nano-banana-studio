# Brand Book Generator Reference

> Load this when the user runs `/create-image book`. Generates a complete visual
> brand book from a brand preset in three formats: Markdown + images,
> PowerPoint (.pptx), and self-contained HTML (print to PDF).

## Usage

```bash
python3 ${CLAUDE_SKILL_DIR}/scripts/brandbook.py generate \
  --preset PRESET_NAME \
  --output ~/brand-book/ \
  --tier standard
```

## Tiers

| Tier | Pages | Images | Use Case |
|------|-------|--------|----------|
| `quick` | 1-2 | 4-6 | Share basics with freelancer or new team member |
| `standard` | 8-15 | 12-18 | Design team reference, agency brief, marketing handbook |
| `comprehensive` | 30+ | 25-35 | Enterprise brand management, global consistency |

## What Gets Generated

### Quick Tier
- Color palette with hex/RGB/CMYK/Pantone codes
- Typography specimen (headline + body)
- 2 photography style samples
- Do's/don'ts (text)

### Standard Tier (includes Quick)
- Cover page with brand name + hero visual
- Full color system with all four color formats
- Typography hierarchy (headline, subheading, body, caption)
- 4 photography samples (product, portrait, editorial, environment)
- Visual motif / pattern samples
- Do's and don'ts with visual examples
- 3 social media templates (Instagram, LinkedIn, YouTube)
- 2 slide templates (title + content)

### Comprehensive Tier (includes Standard)
- Additional photography contexts (food, architecture, abstract)
- Mood board page
- Brand story / values page
- Iconography style samples
- Color application rules (primary vs secondary vs accent)
- Additional social templates (TikTok, Pinterest, Twitter)
- Additional slide types (quote, section divider)

## Output Formats

All three are generated by default (`--format all`):

1. **Markdown + Images** — `brand-book.md` + `images/` folder
2. **PowerPoint** — `brand-book.pptx` (editable in Keynote/PPT/Slides)
3. **HTML** — `brand-book.html` (self-contained, print → Save as PDF)

Use `--format markdown`, `--format pptx`, or `--format html` for a single format.

## Options

| Flag | Description |
|------|-------------|
| `--preset NAME` | Brand preset to use (from `~/.banana/presets/`) |
| `--output PATH` | Output directory |
| `--tier TIER` | `quick`, `standard`, or `comprehensive` (default: standard) |
| `--format FORMAT` | `all`, `markdown`, `pptx`, or `html` (default: all) |
| `--logo PATH` | Path to logo file (placed in brand book, not in AI prompts) |

## Color Specifications

The brand book automatically converts your hex colors to all four formats:

| Format | Use Case | Example |
|--------|----------|---------|
| Hex | Web, digital design | #2563EB |
| RGB | Screen display | (37, 99, 235) |
| CMYK | Print production | (84, 58, 0, 8) |
| Pantone | Color matching, textiles | Pantone 2728 C (nearest) |

Pantone matching is approximate (nearest match by color distance). For critical
print work, verify with a physical Pantone swatch book.

## Tips

- Run `/create-image brand` first to create your preset if you don't have one
- Use `--tier quick` for a fast preview before committing to `standard`
- The HTML output is the best path to PDF — open in browser, Print, Save as PDF
- PowerPoint output is editable — add your own slides, swap images, adjust text
- Logo files are placed as-is (not regenerated) to maintain exact brand identity
